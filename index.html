Travel Spy

Click Here To Go Places

This fun group project explored all of the technologies I have learned up to this point in my education. A user has the ability to search any destination on the planet and view either a live webcam, or timelapses if no live feed is currently available. User authentication is enabled so that individual users can save their favorite search results. The application utlilizes webcam.travel's webcam API, and Geocode API to translate names of locations into coordinates in order to complete the search. I also implemented a flight tracking API to allow the user to easily view current available flights to the currently viewed location from a number of major airlines.

  • Date: May 2019
  • Category: jQuery/Firebase/AJAX
  • GitHub Code: Click Here
